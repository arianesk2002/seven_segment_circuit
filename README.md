# 🔢 Seven Segment Circuit

> A digital logic project for designing and implementing a **seven-segment display decoder/circuit**.  
> This project demonstrates how binary inputs are converted into segment signals to represent digits (0–9) on a 7-segment display.

---

## 📌 Overview

This project focuses on the design and implementation of a **seven-segment display driver circuit**.

It takes binary or encoded inputs and activates the correct segments (`a` to `g`) to display numeric values on a 7-segment display.

> *The main goal is to understand the logic behind display encoding and digital circuit design.*

---

## ✨ Features

- ✔️ Binary / encoded input to 7-segment output conversion  
- ✔️ Digital logic-based design (combinational circuit)  
- ✔️ Supports digits **0–9**  
- ✔️ Segment mapping logic (`a, b, c, d, e, f`)
